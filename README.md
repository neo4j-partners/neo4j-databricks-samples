# Neo4j Databricks Samples

Sample projects for connecting Neo4j to Databricks.

## Projects

| Project | Description |
|---------|-------------|
| [connect-test](connect-test/) | Validates network and driver connectivity to Neo4j Aura — first from your local machine, then from a Databricks cluster |
| [private-link-ee](private-link-ee/) | Private connectivity from Databricks serverless compute to a self-hosted Neo4j Enterprise Edition cluster on Azure, using Azure Private Link |
